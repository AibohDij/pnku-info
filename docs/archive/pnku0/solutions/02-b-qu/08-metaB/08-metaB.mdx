# metaB

## 题面

她和你说，每个人的人生都是有各种各样的元素组成的。<span className="text-red-600">那么，你呢？</span>

## 答案

<AnswerBlock>BACKGROUND</AnswerBlock>

## 解析

这是一个化学主题的区域，而 metaB 也是和化学相关的。七个小题答案分别指向了七种化学元素。

- B1 **BARIUM** (直接翻译即可，元素钡，符号Ba）

- B2 **PENCIL** （铅笔含有的主要元素是碳，所以指向元素碳，符号C）

- B3 **POTASH** （直接翻译即可，钾碱，指向元素钾，符号K）

- B4 **GRAVITONIUM** （在做这道题的途中或许能发现答案，符号Gr）

- B5 **BREATH** （呼吸，指向元素氧，符号O）

- B6 **URANUS** （天王星，是元素铀的词根，符号U）

- B7 **NUMBERSIXTY**（第60号元素，指向元素钕，符号Nd）

得到答案 **【BACKGROUND】**
